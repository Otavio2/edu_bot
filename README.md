# 🤖 Bot IA Universal (Groq + Telegram)

Um bot **avançado** para Telegram, com inteligência artificial da **Groq**, memória de longo prazo, suporte multilíngue, autocuidado, e um **painel administrativo exclusivo para o dono** via botões interativos.

---

## ⚙️ Requisitos

- Python 3.10+
- Conta no Telegram (criar um bot pelo [BotFather](https://t.me/BotFather))
- Conta na [Groq](https://console.groq.com/keys) para gerar sua API Key
- Servidor de deploy (Render ou Heroku recomendado)
